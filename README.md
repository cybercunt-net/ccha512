# ccha-512

Simple Hashing Algorithm by [cybercunt.net](https://cybercunt.net)

## Installation

Download one of the binaries or compile from source

```bash
gcc src/ccha-512.c -o ccha512
```

## Usage

```bash
./ccha512 samplefile.txt
```
